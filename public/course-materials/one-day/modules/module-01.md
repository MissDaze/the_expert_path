## Quick Reference

### Git Commands
```
git init              # Initialize repository
git add .             # Stage changes
git commit -m "msg"   # Create commit
git push              # Push to remote
git pull              # Get latest changes
git branch            # Create/list branches
git merge             # Merge branches
```

### Python Essentials
```python
# Variables
name = "John"
age = 25

# Conditionals
if age >= 18:
    print("Adult")

# Loops
for i in range(5):
    print(i)

# Functions
def greet(name):
    return f"Hello, {name}!"

# Lists
numbers = [1, 2, 3, 4, 5]

# Dictionaries
person = {"name": "John", "age": 25}
```
